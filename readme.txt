Welcome to WUM, the world's first class responsive free website uptime monitoring cloud service. 24/7 monitoring available from around the world, with instant response times and notifications.

== Description ==

WUM Solutions is an external service that allows you to easily monitor your website and receive alerts when your Wordpress website is down, you will also be notified when your website is back up and running.

When you add your website to be monitored by WUM, we will ping your website every 5 minutes, this is to allow us to check if your website is responding.

Your email address will only be used for alerting purposes only and not spam or sold to any third parties.

Please read our [terms and conditions](https://www.httpstatus.co.uk/uptime-monitoring/support/terms-and-conditions) and [privacy policy](https://www.httpstatus.co.uk/uptime-monitoring/support/privacy-policy) before signing up.

== Installation ==

1. Install through Wordpress plugins page.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the WUM menu in the admin area of your site, click the Start Monitoring button, you will be taken to WUM's website where you will need to create an account.
4. When you have created an account on WUM's website and added your site, you will obtain a website key, enter this key into step 2.
